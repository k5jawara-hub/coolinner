📂 LANGKAH 1

Buka File Manager

Masuk ke Control Panel InfinityFree, lalu buka File Manager.

📁 LANGKAH 2

Buat Folder images

Di dalam folder htdocs, buat folder baru bernama images.

⬆️ LANGKAH 3

Upload Foto

Masuk ke folder images, lalu upload semua foto kamu.

✏️ LANGKAH 4

Daftarkan Nama File

Buka file index.html, cari localImages, tambahkan nama file.

💡 Tips: Nama file harus sama persis dengan yang diupload. Gunakan huruf kecil dan tanpa spasi. Contoh: foto1.jpg, liburan.png
